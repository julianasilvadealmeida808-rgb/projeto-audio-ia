# Projeto Assistente de Voz com IA

Este projeto realiza a captura de áudio, transcrição para texto utilizando um modelo de reconhecimento de fala, processamento da pergunta e geração de resposta.

A integração com a API da OpenAI foi implementada, porém a resposta foi simulada devido à limitação de crédito disponível.

## Funcionalidades

- Gravação de áudio
- Transcrição de fala para texto
- Processamento da pergunta
- Geração de resposta
- Conversão da resposta em áudio (gTTS)

## Tecnologias utilizadas

- Python
- Whisper
- gTTS
- Google Colab

## Observação

A resposta da IA está simulada devido à limitação de crédito da API.
